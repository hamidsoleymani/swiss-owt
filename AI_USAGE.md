# AI Usage

## AI Tool Used

- ChatGPT

---

## What AI Was Used For

### Frontend

ChatGPT was used to write:

- Angular routing
- CSS
- HTML
- `proxy.conf.json`
- Angular Signals

### Backend

ChatGPT was used to assist with:

- Code quality improvements
- Dockerfiles
- Overall project folder structure

---

## Representative Prompts

### Prompt 1

```text
Create a Boat model with the following properties:

id?;
name;
description;
createdAt?;
```

### Prompt 2

```text
I do not want a complicated CORS configuration.
Just give me a proxy.conf.json for development.
```

### Prompt 3

```text
I want to use OpenAPI to generate TypeScript types from the backend and then generate Swagger documentation.
Which dependency should I install for the frontend?
```

---

## How the AI Output Was Validated

The generated code was reviewed and tested before being integrated into the project.

For every generated solution, I tried to understand the implementation and asked myself whether I would have implemented it the same way. When necessary, I modified or corrected 
the generated code before using it which did not often happen. 

---

## What Was Not Delegated to AI

Most of the backend implementation was developed without relying on AI.


The core backend implementation and business logic were intentionally implemented manually to ensure a solid understanding of the technologies and design decisions.
